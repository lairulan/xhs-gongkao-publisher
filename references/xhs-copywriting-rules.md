## 标题规则

### 长度
- 严格 ≤ 20 字（小红书标题上限）
- 最佳 12-18 字

### 高点击率标题公式

| 公式 | 示例 |
|------|------|
| 数字 + 痛点 | "行测5个秒杀技巧救了我" |
| 反转/意外 | "双非在职备考竟然一次上岸了" |
| 对比 | "申论从40到65我只做了这一件事" |
| 提问 | "公考到底要不要报班？" |
| 感叹 | "这些神仙岗位没人报太可惜了！" |
| 时间+结果 | "3个月从零基础到行测75" |
| 人群定位 | "在职备考的姐妹看过来" |

### 标题禁忌
- 不用"最全""最好""第一"等绝对化词语
- 不用"内部""泄题""押中"等违规词
- 不做虚假承诺型标题

## 正文规则

### 信息图配文（300-500 字）

```
结构模板：
[钩子] 1-2句吸引注意力
[过渡] 引出图片内容主题
[概括] 简要说明图片讲了什么（不重复图片内容，而是补充/总结）
[互动] 1-2句引导评论
[话题] #话题1 #话题2 ...
```

示例：
```
姐妹们！行测资料分析是不是每次都做不完？😭

我备考的时候也是这样，后来总结了5个速算技巧，真的省了好多时间

图里都整理好了，从估算法到截位法都有👆
觉得有用记得收藏，考前翻出来看！

你们资料分析一般能做对几个？评论区说说～

#公考 #行测 #资料分析 #行测技巧 #备考干货 #公务员考试
```

### 文字笔记正文（500-800 字）

```
结构模板：
[钩子] 第一句必须吸引人（反转/数据/提问/场景）
[正文] 核心内容，3-5个段落
  - 每段2-3句话
  - 段落间空一行
  - 每段有小要点或小标题
[总结] 1-2句总结
[互动] 引导评论的问题
[话题] #话题标签
```

### 排版要求
- **短段落**：每段不超过 3 句话
- **多换行**：段落间空一行
- **要有层次**：用 emoji 做小标题（📌✅💡🔥等）
- **数字突出**：用数字开头的列表（1️⃣2️⃣3️⃣ 或 ①②③）
- **重点加粗**：关键信息可以用【】框起来

## Emoji 使用规范

### 推荐使用
- 段落标记：📌 ✅ 💡 🔥 ⭐ 📖 ✍️
- 情绪表达：😭 🥺 💪 🎉 👏 😤
- 方向指引：👆 👇 ➡️ ⬇️
- 数字序号：1️⃣ 2️⃣ 3️⃣ 4️⃣ 5️⃣

### 使用频率
- 信息图配文：3-5 个 emoji
- 文字笔记：5-10 个 emoji
- **不要**：每句话都带 emoji、连续 3 个以上 emoji

## 话题标签规则

### 数量
- 每篇 5-8 个话题标签
- 放在正文末尾，用空格分隔

### 选择策略
- 2-3 个通用大标签（#公考 #公务员考试 #上岸）
- 2-3 个方向标签（#行测技巧 #申论素材）
- 1-2 个人群标签（#在职备考 #应届考公）

### 格式
- 用 `#` 号开头
- 标签之间用空格分隔
- 放在正文最后一行或倒数第二行

## 互动引导（CTA）

### 高互动 CTA 模板
- 提问型："你们备考多久了？评论区报个数～"
- 征集型："你觉得最难的模块是什么？我下期专门讲"
- 选择型："你是全职备考还是在职备考？"
- 共鸣型："有同款崩溃瞬间的举个手🙋"
- 收藏型："先码住！考前翻出来看"
- 转发型："转给你身边备考的朋友"

### CTA 位置
- 正文倒数第 2-3 行
- 在话题标签之前

## 去 AI 味写作规则（反 AIGC 检测）

> 小红书 AIGC 检测准确率约 95%，核心原理：AI 生成的文本"过于流畅、过于规范、过于理想"。以下规则旨在打破 AI 的文本特征。

### 绝对禁止的 AI 典型模式

| AI 特征 | 禁止示例 | 人味替代 |
|---------|---------|---------|
| 承接词开头 | "首先...其次...最后..." | 直接说内容，删掉承接词 |
| 完美排比 | "不仅...而且...更..." | 打乱句式，长短交替 |
| 总结式结尾 | "总之/综上所述/以上就是..." | "就这些吧"、"先写到这" |
| 过度严谨 | "值得注意的是"、"需要强调" | "有个坑"、"敲重点" |
| 完美因果链 | "因此...所以...由此可见..." | 允许跳跃，直接给结论 |
| 均匀段落 | 每段都是3句话、差不多长度 | 有的段落1句话，有的5句话 |
| 正式书面语 | "建议采取以下方式" | "我觉得这样搞比较好" |

### 必须注入的"人味"元素

每篇内容至少满足以下 4 条（共 7 条）：

1. **至少 2 处私人化场景**：带有时间/地点/人物的具体细节
   - "上个月在图书馆刷真题的时候发现的"
   - "我同事备考那会儿天天抱着这本啃"

2. **至少 1 处不完美表达**：省略号/自我纠正/犹豫/口头语
   - "就是那种...怎么说呢...做完感觉通透了"
   - "不对不对，应该是先做资料分析"

3. **至少 1 处情绪波动**：真实情感而非一直正能量
   - "当时真的要崩溃了谁懂😭"
   - "看到分数的时候我直接蹦起来了哈哈哈"

4. **句式长短交替**：不允许连续 3 句相同长度
   - 短句（5-10字）→ 中句（10-18字）→ 长句（18-25字）交替

5. **至少 1 处话题跳跃**：突然插入的相关但非线性内容
   - "对了说到这个，我突然想起..."
   - "哦差点忘了还有一个"

6. **口语化词汇替换**：至少 3 处书面语→口语
   - "建议" → "我觉得吧"
   - "提高" → "拉上去"
   - "效率" → "速度"
   - "重要" → "贼关键"

7. **非模板化结尾**：最后一句不能是总结或祝福
   - "先写到这 想到再补充"
   - "有啥问题评论区见"
   - "我去吃饭了拜拜"
